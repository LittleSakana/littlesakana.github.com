* `Command + /` : 注释//
* `Command + shift + /` : 注释/\*…\*/ 
* `Command + x` : 剪切行
* `Command + d` : 复制行
* `Command + shift + b` : 快速跳转到方法的实现部分
* `Command + option + l` : 格式化代码
* `Command + f` : 查找文本
* `Command + r` : 替换文本
* `Command + shift + f` : 在指定文件夹查找文本
* `Command + delete` : 删除一行
* `Command + option + t` : 对代码块添加说明注释
* `Command + e` : 最近打开的文件
* `Command + w` : 关闭当前选项卡
* `Command + +` : 展开区块
* `Command + -` : 折叠区块
* `Command + 右箭头` : 光标移到行尾
* `Command + 左箭头` : 光标移到行头
* `Command + shift + 右箭头` : 选中光标到行尾
* `Command + shift + 左箭头` : 选中光标到行头
